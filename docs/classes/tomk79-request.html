private parse_input() : bool

$_GET, $_POST, $_FILES に送られたパラメータ情報を取りまとめ、1つの連想配列としてまとめま、オブジェクト内に保持します。

コマンドラインから実行された場合は、コマンドラインオプションをそれぞれ = 記号で区切り、URLパラメータ同様にパースします。

このメソッドの処理には、入力文字コードの変換(UTF-8へ統一)などの整形処理が含まれます。
